# LLMHelper

[English](README.md) · **한국어**

선택한 텍스트를 로컬 [Ollama](https://ollama.com) 모델한테 넘겨서 번역하거나, 쉽게 풀어주거나, 자세히 설명하게 만든 작은 macOS 메뉴바 도구다. 답은 마우스 옆에 뜨는 작은 창에 한 글자씩 흘러나오고, 그대로 복사하면 된다. 전부 로컬(`localhost:11434`)에서만 돌기 때문에 인터넷도, CORS 설정도, API 키도 필요 없다.

## 기능

- **전역 단축키** — 텍스트를 선택하고 `⌃⌥1`(번역) · `⌃⌥2`(쉽게 설명) · `⌃⌥3`(상세히)을 누르면 끝. 선택한 부분을 알아서 집어온다.
- **메뉴바 방식(권한 불필요)** — 권한 주기 싫을 때 쓴다. 텍스트를 `⌘C`로 복사한 다음 메뉴바 ✨ 아이콘에서 모드를 고르면 된다.
- **스트리밍 플로팅 창** — 커서 옆에 떠서 답을 실시간으로 써 내려간다. **복사** 버튼이나 `⌘C`로 가져갈 수 있고, 다른 데를 클릭하면 알아서 사라진다.
- **모델 선택** — ✨ 메뉴를 열면 설치해둔 Ollama 모델이 쭉 뜬다(임베딩 모델은 빼고). 하나 고르면 바로 적용된다.
- **설정 창** — Ollama 주소, 쓸 모델, 프롬프트 세 개를 직접 고칠 수 있다. 프롬프트 안의 `{text}` 자리에 선택한 글이 들어간다. 마음에 안 들면 기본값으로 되돌리면 된다.

## 요구 사항

- macOS 12 이상
- 로컬에서 실행 중인 [Ollama](https://ollama.com) + 채팅 모델 1개 이상:
  ```bash
  ollama pull qwen3:8b      # 한/영 품질 좋음 (기본값)
  # 더 가볍고 빠른 모델:
  ollama pull qwen2.5:1.5b-instruct
  ```
- 빌드용 Xcode 커맨드라인 도구(`swiftc` / `codesign`)

## 빌드 & 설치

```bash
./build.sh                                   # → build/LLMHelper.app (universal)
cp -R build/LLMHelper.app /Applications/
xattr -cr /Applications/LLMHelper.app        # quarantine 제거
open /Applications/LLMHelper.app             # 메뉴바에 ✨ 아이콘 등장
```

`build.sh`는 머신에 있는 유효한 코드서명 인증서(예: *Apple Development*)를 찾아서 서명한다. 이렇게 해두면 다시 빌드해도 macOS 권한이 풀리지 않는다. 인증서가 없으면 ad-hoc 서명으로 넘어간다. 참고로 서명된 `build/` 결과물은 깃에 올라가지 않는다.

## 권한

**전역 단축키**는 선택한 글을 가져오려고 대신 `⌘C`를 눌러준다. 그래서 **손쉬운 사용(Accessibility)** 권한을 딱 한 번 켜줘야 한다:

> 시스템 설정 ▸ 개인정보 보호 및 보안 ▸ **손쉬운 사용** → **LLMHelper** 켜기

권한이 영 내키지 않으면 메뉴바 방식(`⌘C` 하고 ✨ 클릭)을 쓰면 된다. 이건 권한이 필요 없다.

## 사용법

| 동작 | 단축키 | 메뉴바 |
|---|---|---|
| 번역 (한↔영) | `⌃⌥1` | ✨ → 번역 |
| 쉽게 설명 | `⌃⌥2` | ✨ → 쉽게 설명 |
| 상세히 | `⌃⌥3` | ✨ → 상세히 |

## 설정

✨ ▸ **설정…** 에서 주소와 모델, 각 프롬프트를 바꿀 수 있다. 선택한 글이 들어갈 자리에는 `{text}`를 적어두면 된다. 저장하면 곧바로 반영되고 `~/.config/llmhelper/config.json`에도 남는다.
